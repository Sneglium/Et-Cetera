Et Cetera: API & Utilities Reference

This file is the index for the official Et Cetera API documentation. This does not include mini-APIs provided by modules; every util mentioned here is always enabled, as opposed to module APIs which can be disabled or have unsatisfied dependencies.
Additionally, features that are not considered stable and in a finished state (as well as those that are meant only for internal use) are intentionally left out.
